HTML defines the content and structure of your website

HTML is the standard markup language for Web pages.


With HTML you can create your own Website.

HTML comments are not displayed in the browser, but they can help document your HTML source code.

Short note on tag and element in html with example

The < body > element defines the document's body. It has a start tag body and an end tag . The < h1 > element defines a heading. The < p > element defines a paragraph

What is doctype declaration in html

The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.
